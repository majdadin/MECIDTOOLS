THE PLUGIN IS DESIGND FOR AUTOCAD SPECIFICALLY 
THE MECHANISM ....
1. TYPE GEN-SECTION A MENU WILL APEAR THAT CONTAINS THE INFORMATIONS ABOUT THE DIMENSION AND THE LW OF THE LINES
2. THE AUTOCAD WILL ASK U TO PICK FIRST AND LAST POINT OF THE SECTION THAT IS CUT . EVEN IF THE SECTION LINE IN NONSTRAIGHT ANGEL IT CAN STILL WORK PERFECTLY
3. THERE IS 5 PARTS OF THE REQUIRMENT THAT IS DONE BY ONLY PICKING POINTS FOR 
    1. THE WALL SECTION INTESECTION
    2. THE ELEVATION LINES
    3. THE DOORS
    4. THE WINDOWS FRAME
    5. THE BEAMS
4. THE SECTION WILL BE INSERTED WHERE THE USER CLICK
5. SOME MODIFICATIO WILL STILL BE NEEDE TO COMPLETE IT
THERE IS ALSO THE ADD-LEVEL COMMAND WHICH INSERT A BLOCK THAT I HAVE ALREADY DRAW IT AND CHANGE THE ATT OF IT TO CALCUALTE IT EXACTLY AS IT IS MEASURED MANUALLY
NOT : BOTH THE GEN-SECTION AND ADD-ELEVATION CAMMANDS WORK IN ANY ANGELS WHICH CAN MAKE THE USER BE ABLE TO WORK IN ANY DIRECTION
NOT : THE CODE IS DESIGNED TO WORK FOR ANY PLAN 2D AND IT GENERATES 2D
NOT : THE PROGRAM IS NOT DESIGNED TO WORK ON A SPECIAL ELEMENTS. IT WORKS MANUALLY
